# 隐私说明

`THEOL 课程资源批量下载` 扩展不会收集、存储、出售、共享或上传任何个人数据到开发者服务器或第三方分析平台。

## 扩展如何工作

- 扩展仅在匹配的 THEOL 课程资源页面中运行。
- 扩展读取当前页面的课程资源目录信息，并在当前登录会话下向同源 THEOL 站点发起文件下载请求。
- 下载的数据直接保存到用户浏览器下载目录、浏览器创建的 ZIP 文件，或用户明确选择的本地目录。

## 数据处理范围

- 不采集账号、密码、Cookie、学号、姓名、成绩等身份数据。
- 不建立远程日志、分析、遥测、崩溃上报或广告跟踪。
- 不把课程资源内容发送到开发者控制的服务器。
- 不读取与 THEOL 课程资源页面无关的网站内容。

## 权限说明

- `downloads`：用于批量创建浏览器下载任务。
- 站点匹配权限 `*://*/meol/*`：用于在不同学校部署的 THEOL 课程资源页面中注入扩展，并在用户已登录的前提下访问同源资源。

## Firefox 数据声明

对于 Firefox `data_collection_permissions`，本扩展声明为：

- `required: ["none"]`

即扩展不要求收集或传输任何用户数据到扩展外部。
