# 扩展商店提交材料

适用对象：

- Chrome Web Store
- Microsoft Edge Add-ons
- Firefox Add-ons

## 基本信息

- 扩展名称：`THEOL 课程资源批量下载`
- 英文名称：`THEOL Batch Downloader`
- 版本：`3.0.0`
- 类别建议：`Productivity`
- 目标用户：使用 THEOL / MEOL 课程平台的高校师生

## 简短描述

用于在 THEOL 课程资源页面中递归扫描目录、批量勾选文件，并按原有层级下载。

## 详细描述

`THEOL 课程资源批量下载` 用于减少高校在线课程平台中逐个点击下载文件的重复操作。扩展会在 THEOL 课程资源页面中扫描当前目录及其子目录，生成可勾选的资源树，并提供两种下载方式：

- 打包为 ZIP 下载
- 按原目录结构批量创建下载任务

主要能力：

- 自动递归扫描当前课程资源目录
- 树状结构展示课程文件和子目录
- 支持按目录全选、展开、收起
- 下载前估算可统计文件的总大小
- 基于响应头和图标自动补全常见文件后缀
- 尽量保留课程资源原始目录结构

扩展只在匹配的 THEOL 页面中运行，不绕过登录、不提升权限，也不会把课程内容上传到开发者服务器。

## 权限使用说明

### `downloads`

用于批量创建浏览器下载任务。没有此权限，扩展无法自动下载用户勾选的多个课程文件。

### 站点权限 `*://*/meol/*`

不同学校部署的 THEOL / MEOL 平台通常使用不同的域名和二级域名。扩展需要在这些课程资源页面中运行内容脚本，才能识别当前目录结构、列出文件并在用户已有登录状态下发起同源下载请求。

该权限仅用于 THEOL / MEOL 页面，不用于读取其它网站内容。

## 审核备注

该扩展仅在高校 THEOL / MEOL 课程资源页面内提供批量下载辅助功能：

- 不包含账号登录、验证码处理、自动化绕过或权限提升逻辑
- 不连接开发者服务器
- 不上传用户课程内容
- 所有下载请求都发往用户当前已登录的学校 THEOL 站点

如果审核人员需要验证功能，可在任意兼容的 THEOL / MEOL 测试环境中打开课程资源页面后进行检查；扩展本身不依赖外部 API。
